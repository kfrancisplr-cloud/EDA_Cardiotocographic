# EDA - Cardiotocographic Dataset

## Objective
To analyze fetal health data using Exploratory Data Analysis (EDA).

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Work Done
- Data cleaning (handled missing values using median)
- Removed duplicates
- Outlier detection using IQR
- Visualizations (histogram, boxplot, heatmap, etc.)
- Found relationships between variables

## Conclusion
EDA helped understand patterns in fetal health data and can be used for further prediction.# EDA_Cardiotocographic
